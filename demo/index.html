Progress bar

Progress bar with custom bounds

By default value is clamped between 0 and 1. You can customize this bounds using min and max attributes.

Warning! Min attribute must be less then max.

Indeterminate Progress Bar

When the value is undefined the progress bar will animate indefinitely to show that progress is ongoing.

Indeterminate Progress Circle
